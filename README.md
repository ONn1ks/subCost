# subCost — трекер расходов на подписки

Веб-приложение помогает собирать все подписки в одном месте, рассчитывать фактические ежемесячные и годовые траты, следить за датами списаний и пополнять список популярными сервисами в один клик. Интерфейс выполнен на React + TypeScript (Vite).

## Основные возможности

- **Каталог топовых сервисов** — 30 международных и 30 российских подписок с описанием, средней стоимостью и валютой.
- **Добавление своих подписок** — имя, стоимость, периодичность (день, неделя, месяц, год), валюта, день списания, заметки и загрузка изображения.
- **Статистика** — суммарные траты в месяц и год, средний чек, разрез по периодичности и источникам, список самых дорогих подписок.
- **Фильтры и поиск** — быстрый поиск по названию или заметкам, фильтрация по типу подписки и периодичности.
- **Локальное хранение** — данные сохраняются в `localStorage` браузера и остаются только у пользователя.

## Запуск проекта

```bash
npm install
npm run dev
```

После старта dev-сервера приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173).

### Сборка и проверка

```bash
npm run build    # типизация + production-сборка
npm run lint     # проверка кода ESLint
```

## Структура проекта

```
src/
├── App.tsx                # композиция страницы и состояние приложения
├── components/            # UI-компоненты (форма, список, статистика, каталог)
```

## Перспективы развития

- экспорт и импорт данных, синхронизация с облаком;
- уведомления о скором списании средств;
- расширенные графики и прогнозы по изменениям подписок.

Приложение создано для демонстрации подхода к управлению подписками и может служить основой для дальнейшей разработки полноценного сервиса.
