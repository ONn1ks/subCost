# SubCost

Интерактивный веб-сервис для учёта цифровых подписок. Позволяет импортировать собственные данные, добавлять новые сервисы, загружать обложки и получать наглядную аналитику по расходам.

## Возможности

- Готовые подборки топ-30 международных и российских подписок с быстрым добавлением в личный список.
- Форма добавления с поддержкой разных валют, настраиваемой периодичности и прикреплением изображений.
- Автоматический пересчёт ежемесячных и годовых затрат, средних значений и статуса бюджета.
- Отображение ближайших списаний, диаграммы расходов по категориям и текстовых инсайтов.
- Экспорт и импорт данных в формате JSON, а также шаблон CSV для массового ввода.
- Хранение данных локально в браузере, без сервера.

## Как запустить

1. Склонируйте репозиторий и откройте его в любой статический сервер или просто запустите `index.html` в браузере.
2. Для локального сервера можно воспользоваться встроенным в VS Code Live Server или командой `python -m http.server` из корня проекта.
3. Откройте страницу в браузере и начинайте добавлять подписки.

## Импорт и экспорт

- **Экспорт** создаёт JSON-файл с текущими настройками и подписками.
- **Импорт** принимает JSON в формате, который возвращает экспорт.
- Для первичного заполнения можно скачать CSV-шаблон, заполнить его и по данным добавить подписки вручную.

## Стек

- Чистый HTML, CSS и JavaScript без сборщиков и зависимостей.
- Chart.js для визуализации распределения расходов по категориям.
